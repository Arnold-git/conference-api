# Talk API 

## Services
- Talk
  - create a talk
  - add attendee to a talk
  - delete a talk
  - get a talk
  - get all talks
- Attendee
  - create an attendee
  - get an attendee
  - delete an attendee
- Chat 
    - create a message


## Set Up 
```$ npm install ```
# development
``` $ npm run start ```

# watch mode
``` $ npm run start:dev ```
